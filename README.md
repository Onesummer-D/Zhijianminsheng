# 智检民声 —— 12345涉检线索智能筛查系统

[![Python](https://img.shields.io/badge/python-3.8+-green.svg)](https://www.python.org/)
[![Gradio](https://img.shields.io/badge/Gradio-5.x-orange.svg)](https://gradio.app/)

> 基于双层级权重规则引擎 + DeepSeek API 语义兜底的多标签分类系统，面向房山区人民检察院 12345 工单涉检线索智能筛查需求。

---

## 一、项目背景

12345 政务服务便民热线日均产生大量民生投诉，其中隐藏着涉检线索（刑事犯罪、民事支持起诉、行政执法监督、公益诉讼）。传统人工筛查效率低、标准不一、易遗漏。

"智检民声"系统通过 NLP 技术实现：

- **智能分类**：将口语化投诉自动映射到"四大检察"业务类别
- **法条推荐**：实时关联适用法律依据
- **相似案例**：基于 128 条典型案例库进行伪 RAG 匹配
- **点位提取**：自动识别房山区地址信息，支撑公益诉讼点位整合
- **重复聚合**：识别同一主体、同一地点的重复投诉
- **数据脱敏**：自动遮蔽公民隐私信息

---

## 二、核心功能

| 功能模块 | 说明 |
|---------|------|
| 单条查询 | 输入单条 12345 工单，实时返回类别、置信度、法条、相似案例、点位、要素提取 |
| 批量筛查 | 上传 Excel 批量分析，支持进度条、导出结果、筛查台账 |
| 法条关联 | 自动匹配《刑法》《民法典》《行政诉讼法》等相关法条 |
| 点位识别 | 四层清洗正则提取"区-镇-村"地址结构 |
| 相似案例 | TF-IDF + Min-Max 归一化，返回 60%-100% 相似度案例 |
| 数据脱敏 | 自动遮蔽姓名、手机号、身份证号、银行卡号、门牌号 |
| 审计日志 | 每次查询自动记录，支持后续溯源分析 |

## 四、快速开始

### 1. 环境要求

- Python 3.8+
- pip

### 2. 安装依赖

```bash
pip install gradio pandas openpyxl jieba scikit-learn requests
```

### 3. 配置 API Key

```bash
export DEEPSEEK_API_KEY="your-api-key-here"
```

> 请勿将 API Key 硬编码在代码中！

### 4. 启动服务

```bash
python app.py
```

服务启动后访问：`http://localhost:6009`

## 六、核心算法说明

### 6.1 双层级权重算法

| 层级 | 权重 | 示例 |
|------|------|------|
| 核心定性词 | +3 | "挪用资金""恶意欠薪""偷排污水" |
| 特征词 | +1 | "拖欠工资""罚款太重""垃圾倾倒" |

**置信度分级**：>=6 分（高）、3-5 分（中）、<3 分（低）

### 6.2 融合决策策略

```
if 规则引擎高分（>=6 且分差>=3）:
    直接返回规则结果（快速通道，无API调用）
elif 规则引擎中低分或存在交叉:
    调用 DeepSeek API 语义兜底（复核通道）
else:
    判定为普通投诉（排除词保护门阻断API）
```

### 6.3 准确率表现

基于 108 条房山区真实工单测试集：

|--------|---------|---------|
